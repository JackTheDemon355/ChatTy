# ChatTy Ultra+

ChatTy Ultra+ is an advanced, real‑time chat platform featuring:

- Google Sign‑In (GIS)
- GitHub OAuth
- Discord OAuth
- Google‑only rooms
- Google‑verified rooms
- Global admin system
- Host controls
- Direct messages
- Friends system
- File sharing
- Typing indicators
- Responsive UI
- Theming (Light / Dark / Blue)

This project is **open‑source** under the **GNU General Public License v3.0 (GPL‑3.0)**.

---

## 🚀 Features

### Authentication
- Google Identity Services (no secret required)
- GitHub OAuth (Passport)
- Discord OAuth (Passport)
- Multi‑account switching
- OAuth profile syncing

### Rooms
- Create / join rooms
- Password‑protected rooms
- Google‑only rooms
- Google‑verified rooms
- Host transfer
- Lock / unlock rooms
- Clear chat
- Change room password

### Admin
- Global admin roles
- Make/remove admin
- Global kick
- Host controls (mute, unmute, kick, ban)

### Chat
- Real‑time messaging
- Typing indicators
- File sharing
- Avatar + name editing
- Google / GitHub / Discord badges

### DM & Friends
- Direct messages
- Friends list
- Add/remove friends
